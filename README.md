# My_Day_App_PWA
Тестовое приложение "Трекер Привычек" для мобильных устройств

# Инструкция по установке
Выгруженное приложение находится здесь: https://life-tracker-wd13.onrender.com

Чтобы вы смогли установить приложение, нужно зайти на него в случае с: 

a)	Android – через chrome, нажать 3 точки, там нужно добавить на главный экран. Если все сделано правильно, то браузер откроет окно установки. Нужно будет согласиться, и приложение установится на смартфон. При открытии оно будет открываться как отдельное приложение.

b)	iOS – через safari. Нажимаем на значок «поделиться» и нажимаем на «показать еще», а потом на «добавить на экран домой». Если есть переключатель «Открывать как веб приложение», то тыкаем на него.
